Installation

Démarrage en 3 commandes

1
Cloner
Récupérer le projet depuis GitHub sur votre serveur local.
2
Configurer
Renseigner l'IP de votre onduleur dans l'onglet Paramètres.
3
Lancer
Docker Compose démarre nginx + sidecar Python automatiquement.
# Cloner le repo
git clone https://github.com/struppy/indevolt-dashboard.git
cd indevolt-dashboard

# Créer le dossier de données
mkdir -p data/store

# Lancer
docker compose up -d

# Ouvrir dans le navigateur
http://VOTRE_IP:8480
